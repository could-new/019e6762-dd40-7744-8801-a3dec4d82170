# Interactive Resume

A beautiful, interactive, cross-platform resume built with Flutter.

## Overview
This interactive resume application showcases professional experience, education, skills, and projects in a responsive layout that works beautifully on mobile, tablet, desktop, and web. It features a clean design with sections for contact information, summary, work experience, education, and skills.

## Features
- **Responsive Design**: Adapts seamlessly to different screen sizes, displaying a mobile-friendly view on narrow screens and a more expansive layout on desktop.
- **Professional Layout**: Organized sections for Summary, Experience, Education, and Skills.
- **Interactive Elements**: Scrollable views and clean typography.
- **Cross-Platform**: Runs natively on iOS, Android, Web, macOS, Windows, and Linux.

## Tech Stack
- **Framework**: Flutter
- **Language**: Dart
- **Layouts**: Responsive layout with `LayoutBuilder`, `ListView`, and `Card` widgets.

## Getting Started

### Prerequisites
- Install [Flutter](https://flutter.dev/docs/get-started/install)
- A supported IDE (VS Code, Android Studio, IntelliJ)

### Running the App
1. Clone the repository or download the source code.
2. Navigate to the project directory.
3. Run `flutter pub get` to install dependencies.
4. Run the app using:
   ```bash
   flutter run
   ```
